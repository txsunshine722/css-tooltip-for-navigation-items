# CSS Tooltip for Navigation Items

In this project, you create a tooltip that appears above navigation items on hover, using only HTML and CSS.

A tooltip is a small pop-up box that provides additional information about a navigation item when a user hovers over it. This project focuses on mastering CSS positioning, hover effects, and creating visually appealing tooltips without JavaScript.

## Project URL

https://roadmap.sh/projects/tooltip-ui

## Goal

The goal of this project is to help you understand how to use CSS for dynamic UI effects. You will practice how to:

- Position elements relative to each other
- Create smooth tooltip transitions
- Make navigation more interactive and user-friendly

## What You Will Learn

After completing this project, you will have a better understanding of:

- CSS positioning (`relative`, `absolute`, stacking context)
- Hover and focus-based UI behavior
- Building interactive UI elements without JavaScript

## Requirements

- Use only HTML and CSS
- Show a tooltip when a navigation item is hovered
- Position the tooltip above the relevant navigation item
- Keep tooltip content clear and easy to read

## Bonus Ideas

Try adding different tooltip animation styles, such as:

- Fade-in
- Slide-in
- Scale-in

## Getting Started

1. Clone or download this repository.
2. Open the project folder in VS Code.
3. Open `index.html` in your browser.
4. Hover navigation items to test tooltip behavior.

## Tech Stack

- HTML5
- CSS3

## Notes

This project intentionally avoids JavaScript so you can strengthen your CSS fundamentals for interaction and UI behavior.
